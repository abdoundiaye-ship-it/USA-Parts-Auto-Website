# USA Cars Oil & Parts Website

A modern, responsive website for USA Cars Oil & Parts, specializing in vehicle engine oils, automotive supplies, and spare parts imported from the United States.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Contact form for customer interaction
- Product catalog with actual products from the ERP system

## Files

- `index.html`: Main HTML structure with updated products section
- `style.css`: CSS styles for responsive design
- `app.js`: JavaScript for form handling and smooth scrolling
- `Logo - USA Car and Parts.png`: Company logo
- `USA_PARTS_AUTO_pro_stock_erp.xlsx`: ERP workbook used for product data
- Other documents: Project descriptions and branding materials

## Usage

1. Open `index.html` in a web browser to view the website.
2. Navigate through the sections using the menu.
3. Fill out the contact form to send a message.
4. Browse the products catalog for available items.

## Customization

- Update the logo by replacing `Logo - USA Car and Parts.png`.
- Modify text and content in `index.html`.
- Adjust styles in `style.css`.
- Enhance functionality in `app.js`.
- Add more products by editing the product-grid in `index.html`.

## Troubleshooting

- If images don't load, ensure the file paths are correct.
- For form submission, integrate with a backend service to handle emails or database storage.
- Test responsiveness by resizing the browser window or using developer tools.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

## Data Source

Product information was extracted from `USA_PARTS_AUTO_pro_stock_erp.xlsx` to populate the products section.
